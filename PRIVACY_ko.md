# 개인정보처리방침

**최종 업데이트:** 2026년 3월 9일

## 서문

Temporary Bookmarks ("임시 북마크", "당사", 또는 "확장 프로그램")는 귀하의 개인정보를 존중하며 보호하기 위해 최선을 다합니다. 이 개인정보처리방침은 확장 프로그램이 귀하의 정보를 수집, 사용, 보호하는 방법을 설명합니다.

## 데이터 수집

### 1. 로컬 저장 데이터

확장 프로그램은 Chrome 저장소 API를 사용하여 귀하의 기기에 다음 데이터를 로컬로 저장합니다:

- **북마크 데이터:** URL, 페이지 제목, 생성 타임스탬프, 만료 설정
- **사용자 설정:** 기본 만료 일수, 중복 저장 설정, 알림 설정
- **동기화 상태:** 연결 상태, 마지막 동기화 시간 (Google Drive 동기화가 활성화된 경우에만)
- **삭제 대기 중인 항목:** 실행 취소 대기 중인 임시 삭제 북마크 (3초 후 자동 삭제)

**저장 위치:** 모든 데이터는 `chrome.storage.local`을 사용하여 귀하의 기기에 로컬로 저장되며 당사 서버로 전송되지 않습니다.

### 2. Google Drive 동기화 (선택 사항)

Google Drive 동기화를 활성화하는 경우:

- 확장 프로그램은 Chrome Identity API와 OAuth2를 사용하여 안전하게 인증합니다
- 북마크 데이터는 귀하의 개인 Google Drive 계정에 저장됩니다
- `https://www.googleapis.com/auth/drive.file` 권한을 사용하며, 이 확장 프로그램이 생성한 파일에만 접근할 수 있습니다
- **당사는 귀하의 Google Drive 자격 증명이나 데이터에 접근하지 않습니다**

### 3. 브라우저 탭

확장 프로그램은 브라우저 탭에 접근하여 다음을 수행합니다:

- "페이지 저장" 버튼 클릭 시 현재 페이지의 URL과 제목 가져오기
- 이 데이터는 북마크 생성용으로만 사용되며 다른 곳으로 전송되지 않습니다

## 데이터 사용

당사는 수집된 데이터를 다음 목적으로만 사용합니다:

1. 북마크를 로컬에 저장 및 관리
2. Google Drive를 통한 기기 간 북마크 동기화 (활성화된 경우)
3. 만료 설정에 따른 만료된 북마크 자동 삭제
4. 삭제된 북마크의 실행 취소 기능 제공 (5초간)

## 데이터 공유

**당사는 마케팅 목적으로 제3자에게 귀하의 개인 데이터를 판매, 대여, 공유하지 않습니다.**

귀하의 북마크 데이터는 다음과 같은 곳과만 공유됩니다:

- Google Drive (명시적으로 동기화를 활성화한 경우) - 귀하의 개인 계정에 저장
- 그 외 제3자는 데이터를 수신하지 않습니다

## 데이터 보관

- **로컬 저장:** 귀하가 삭제하거나 만료 설정에 따라 만료될 때까지 북마크가 저장됩니다
- **삭제 대기 중:** 임시 삭제된 북마크는 실행 취소를 위해 5초간 보관됩니다
- **Google Drive:** 동기화를 연결하거나 북마크를 삭제할 때까지 Google Drive에 보관됩니다

## 귀하의 권리

귀하는 다음과 같은 권리를 가집니다:

1. **접근권:** 확장 프로그램 팝업에서 모든 북마크 확인 가능
2. **삭제권:** 개별 북마크 삭제 또는 전체 데이터 삭제 가능
3. **내보내기:** Google Drive에서 직접 데이터 접근 가능 (동기화 활성화 시)
4. **동기화 비활성화:** 언제든지 Google Drive 연결 해제 가능

모든 로컬 데이터를 삭제하려면:

1. Chrome 확장 프로그램 페이지 열기 (`chrome://extensions`)
2. "Temporary Bookmarks" 찾기
3. "제거" 클릭 또는 Chrome 설정에서 사이트 데이터 삭제

## 보안

당사는 귀하의 데이터를 보호하기 위해 합리적인 보안 조치를 취합니다:

- 모든 데이터는 Chrome의 보안 저장소 API를 사용하여 로컬에 저장됩니다
- Google Drive 동기화는 OAuth2를 사용하여 안전하게 인증됩니다
- 확장 프로그램은 민감한 개인 정보를 수집하거나 전송하지 않습니다
